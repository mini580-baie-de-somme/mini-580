# Guide équipe Mini5.80 Baie de Somme — Comment publier sur le site

> À partager dans le groupe Telegram #Mini580-Chantier

## C'est simple

Vous n'avez **rien à installer**, **aucun site à ouvrir**, **aucun compte à créer**.

### Pour publier des photos de chantier

1. Ouvrez le groupe **#Mini580-Chantier** sur Telegram
2. Envoyez vos photos (1 ou plusieurs)
3. Ajoutez une petite description, par exemple :
   - *"Pose des couples sur la 268"*
   - *"Livraison contreplaqué Allin"*
   - *"Première séance CNC Rodolphe"*

C'est tout. Le reste est automatique.

### Pour partager un plan, un PDF ou des photos lourdes

1. Ouvrez le dossier Google Drive **Mini5.80 Baie de Somme → À publier**
2. Déposez votre fichier dans le bon sous-dossier (ou créez-en un)
3. Optionnel : envoyez un message dans Telegram pour prévenir

### Bonnes habitudes (pas obligatoires)

| Info utile | Exemple |
|------------|---------|
| Numéro de coque | 268, 269 ou 270 |
| Étape chantier | couples, plaquage, fournisseurs, CNC… |
| Votre prénom | pour qu'on sache qui publie |

### Questions & commentaires

- Posez vos questions directement dans le groupe
- Répondez aux fils de discussion — vos réponses peuvent devenir du contenu sur le site

### Ce que vous n'avez PAS à faire

- Écrire en anglais (traduction automatique)
- Redimensionner les photos
- Se connecter à un site web
- Utiliser un ordinateur (le téléphone suffit)

---

*En cas de problème, contactez l'admin du groupe.*
